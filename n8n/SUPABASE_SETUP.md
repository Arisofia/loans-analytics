# Supabase Setup Guide

Comprehensive setup guide for integrating Supabase with Abaco Loans Analytics.

## Quick Start

1. Create Supabase project at https://supabase.com
2. Copy API keys from Settings > API
3. Store credentials in Azure Key Vault (aiagent-secrets-kv)
4. Connect to Power BI and Grafana using PostgreSQL connection

For detailed instructions, see full documentation in /docs directory.

## Credentials to Store

- supabase-url
- supabase-anon-key  
- supabase-service-role-key
- supabase-db-password
